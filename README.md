# P2C

## 描述

P2C，Physical to Container，一款開源容器化工具，它旨在簡化容器化流程、節省時間並減少複雜性，將物理環境中的應用程式或網站直接打包為容器，使開發人員和運維團隊能夠輕鬆將應用程式或網站部署到雲端，增加可攜性並發揮容器的優勢。

## 安裝

```bash
git clone https://github.com/a3705280/p2c.git
cd src/core
make all
```

## 使用方法

```bash
p2c -help
```
